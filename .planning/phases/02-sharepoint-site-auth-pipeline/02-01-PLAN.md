       This file is committed. The actual `scripts/config.json` (with real tenant values) is gitignored.

    2. Create `scripts/Deploy-WissensHub.ps1` -- the main orchestrator:
       - Accepts `-ConfigPath` parameter (defaults to `./scripts/config.json`)
       - Loads config via `Get-Content $ConfigPath | ConvertFrom-Json`
       - Validates config has required fields (tenantName, siteUrl, adminUpn)
       - Connects to PnP Online: `Connect-PnPOnline -Url $config.siteUrl -Interactive`
       - Dot-sources and calls each module function in order: Site, Groups, Columns, Pages, Navigation, SampleData
       - The Entra ID module call is present but commented out with a note: "Uncommented by Plan 02-02 which creates this module"
       - Wraps each module call in try/catch with colored status output
       - Disconnects PnP at the end: `Disconnect-PnPOnline`
       - Prints summary of what was created/skipped at the end

    3. Create `scripts/modules/New-WissensHubSite.ps1`:
       - Function: `New-WissensHubSite` accepting $Config parameter
       - Idempotent: Check `Get-PnPTenantSite -Url $Config.siteUrl -ErrorAction SilentlyContinue`
       - If not exists: `New-PnPSite -Type CommunicationSite -Title "WissensHub" -Url $Config.siteUrl -Lcid 1031 -Description "Zentrale Wissensdatenbank"`
       - After creation, reconnect to the new site URL: `Connect-PnPOnline -Url $Config.siteUrl -Interactive`
       - Write-Host status messages with color (Green=created, Yellow=skipped)

    4. Create `scripts/modules/New-WissensHubGroups.ps1`:
       - Function: `New-WissensHubGroups` accepting $Config parameter
       - Creates 4 groups from config: Members (Read), Editors (Edit), Reviewers (Read), Owners (Full Control)
       - Idempotent: Check `Get-PnPSiteGroup | Where-Object { $_.Title -eq $groupName }` before creating
       - Use `New-PnPSiteGroup -Name $name -PermissionLevels $permLevel`

    5. Create `scripts/modules/New-WissensHubColumns.ps1`:
       - Function: `New-WissensHubColumns` accepting $Config parameter
       - Creates 6 columns on "Site Pages" library, all with WH_ prefix and Group "WissensHub":
         a. WH_Category (Choice) -- choices from $Config.categories
         b. WH_Status (Choice) -- choices: "Draft", "InReview", "Published", "Archived"
         c. WH_TargetGroups (Note/multi-line text) -- for semicolon-delimited target group names
         d. WH_IsMandatory (Boolean)
         e. WH_Reviewer (User)
         f. WH_ReviewByDate (DateTime)
       - Idempotent: Check `Get-PnPField -List "Site Pages" -Identity $internalName -ErrorAction SilentlyContinue` before each
       - Use `Add-PnPField -List "Site Pages" -DisplayName $display -InternalName $internal -Type $type -Group "WissensHub"`

    6. Append to root `.gitignore`:
       - Add `# Provisioning secrets` section
       - Add `scripts/config.json` (actual config with real tenant values)
